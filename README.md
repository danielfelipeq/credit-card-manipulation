# Proyecto de ejemplo para validar tarjetas de crédito y generar tokens

Este proyecto es un ejemplo de cómo validar tarjetas de crédito y generar tokens.

## Requisitos

* Node.js >= 16.0.0
* npm >= 8.0.0

## Instalación

npm install


## Ejecución

npm start


El proyecto se ejecutará en el puerto 3000.
